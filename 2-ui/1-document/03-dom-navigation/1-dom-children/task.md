# DOM ਬੱਚੇ

ਇਸ ਪੇਜ ਨੂੰ ਵੇਖੋ:

```html
<html>
<body>
  <div>Users:</div>
  <ul>
    <li>John</li>
    <li>Pete</li>
  </ul>
</body>
</html>
```

ਹੇਠ ਲਿਖਿਆਂ ਵਿੱਚੋਂ ਹਰੇਕ ਲਈ, ਉਹਨਾਂ ਤੱਕ ਪਹੁੰਚਣ ਦੇ ਤਰੀਕੇ ਦਾ ਘੱਟੋ ਘੱਟ ਇੱਕ ਢੰਗ ਦਿਓ:
-  `<div>`DOM ਨੋਡ?
-  `<ul>` DOM ਨੋਡ?
- ਦੂਜਾ `<li>` (ਪੀਟ ਨਾਲ)?
